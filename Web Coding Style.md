# Coding Style
 
## Folder Name & File Nam
 > 名稱都小寫，並用“_”區隔單字
 > ex : home_page
 > 如果名稱是縮寫，則全部大寫
 > ex : FAQ

## Parameter
 > 名稱以駝峰式取名
 > ex : homePageIndex

## Function ( Component )
 > 名稱以駝峰式取名，並在開都加上“_”，並都用箭頭函數方式宣告
 > ex : _showThing = () => {doSomething...}

## Function ( General )
 > 名稱以駝峰式取名，並都用箭頭函數方式宣告
 > ex : starIcon = () => {doSomething...}

## Image Path
 > 圖片路徑變數名稱一律大寫，開頭為“IMGPATH_”
 > ex : IMGPATH_LOGO

## CSS Class Name ( General )
 > 名稱一律小寫，並用“-”隔開
 > ex : .slider-page

## CSS Class Name ( Common )
 > 名稱單字第一個字大寫，並用“-”隔開
 > ex : .Main-Page
